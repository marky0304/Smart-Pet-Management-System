# 健康管理模块测试指南

## 前提条件
- ✅ 后端服务运行在 http://localhost:8080
- ✅ 前端服务运行在 http://localhost:3000
- ✅ 数据库已初始化

## 测试步骤

### 1. 登录系统
1. 访问 http://localhost:3000
2. 使用管理员账号登录：
   - 用户名：`admin`
   - 密码：`admin123`

### 2. 添加宠物（如果还没有）
1. 点击左侧菜单"宠物管理"
2. 点击"添加宠物"按钮
3. 填写宠物信息：
   - 名称：例如"小白"
   - 类型：选择"狗"或"猫"
   - 品种：例如"金毛"
   - 性别：选择"公"或"母"
   - 出生日期：选择日期
   - 其他信息可选填
4. 点击"确定"保存

### 3. 测试健康记录功能

#### 3.1 添加体检记录
1. 点击左侧菜单"健康管理"
2. 点击"添加健康记录"按钮
3. 填写信息：
   - 宠物：选择刚才添加的宠物
   - 记录类型：选择"体检"
   - 记录日期：选择当前日期时间
   - 体重：例如 25.5 kg
   - 体温：例如 38.5 ℃
   - 症状描述：例如"定期体检，无异常"
   - 诊断结果：例如"健康状况良好"
   - 医院名称：例如"宠物医院"
   - 医生姓名：例如"张医生"
   - 费用：例如 200 元
4. 点击"确定"保存

#### 3.2 添加疫苗记录
1. 再次点击"添加健康记录"
2. 填写信息：
   - 宠物：选择同一只宠物
   - 记录类型：选择"疫苗"
   - 记录日期：选择日期
   - 诊断结果：例如"狂犬疫苗接种"
   - 用药记录：例如"狂犬疫苗 1针"
   - 下次复诊日期：选择一个月后的日期
   - 费用：例如 100 元
3. 点击"确定"保存

#### 3.3 添加体重记录
1. 添加多条体重记录（不同日期）
2. 记录类型选择"体重记录"
3. 填写不同的体重值，例如：
   - 第1条：25.0 kg
   - 第2条：25.5 kg
   - 第3条：26.0 kg
   - 第4条：26.5 kg

#### 3.4 添加异常体温记录
1. 添加一条记录
2. 体温填写：40.0 ℃（超过正常范围）
3. 症状：例如"发烧，精神不振"
4. 保存后观察是否被标记为异常

### 4. 测试查询和筛选
1. 使用宠物筛选：选择特定宠物
2. 使用类型筛选：选择"疫苗"
3. 使用日期范围筛选：选择最近一周
4. 点击"查询"按钮
5. 点击"重置"按钮清空筛选条件

### 5. 测试健康趋势图表
1. 在筛选区域选择一只宠物
2. 观察页面下方出现的趋势图表
3. 查看"体重趋势"图：
   - 应该显示折线图
   - X轴为日期，Y轴为体重
4. 查看"体温趋势"图：
   - 应该显示折线图
   - 有两条红色标线（37.5℃ 和 39.5℃）
   - 超出范围的点应该明显可见

### 6. 测试记录操作
1. 点击某条记录的"查看"按钮
   - 查看详细信息
2. 点击"编辑"按钮
   - 修改某些信息
   - 保存修改
3. 点击"删除"按钮
   - 确认删除
   - 记录应该从列表中消失

### 7. 测试管理员功能
1. 点击左侧菜单"健康数据管理"
2. 查看统计面板：
   - 总记录数
   - 异常数据数量（应该包含体温40℃的记录）
   - 待复诊数量（应该包含设置了复诊日期的记录）
   - 记录类型分布饼图
3. 查看记录列表：
   - 异常体温应该显示为红色
   - 可以查看所有用户的健康记录
4. 测试筛选功能

### 8. 测试分页功能
1. 如果记录超过10条，测试分页
2. 切换每页显示数量（10/20/50）
3. 切换页码

## 常见问题

### 1. 图表不显示
- 检查是否选择了宠物
- 检查该宠物是否有健康记录
- 检查浏览器控制台是否有错误

### 2. 异常数据不显示为红色
- 确认体温值 < 37.5 或 > 39.5
- 刷新页面重试

### 3. 无法添加记录
- 确认已登录
- 确认已有宠物
- 检查必填字段是否填写

### 4. 管理员功能看不到
- 确认使用管理员账号登录
- 检查左侧菜单是否有"健康数据管理"
