# 服务预约模块说明

## 功能概述

服务预约模块为用户提供在线预约宠物服务的功能，包括洗澡、美容、寄养、医疗等服务。管理员可以管理服务项目、处理预约订单并查看统计数据。

## 用户功能

### 1. 服务浏览与预约
- **服务分类**：洗澡、美容、寄养、医疗
- **服务信息**：
  - 服务名称
  - 服务描述
  - 价格
  - 时长
  - 服务图片
  
- **在线预约**：
  - 选择服务项目
  - 选择宠物
  - 选择预约时间
  - 填写备注信息
  - 自动生成订单号

### 2. 我的预约管理
- **预约列表**：
  - 查看所有预约记录
  - 按状态筛选（待确认、已确认、已完成、已取消）
  - 分页展示
  
- **预约操作**：
  - 查看预约详情
  - 取消预约（待确认/已确认状态）
  - 评价服务（已完成状态）
  
- **评价功能**：
  - 1-5星评分
  - 文字评价

### 3. 预约状态
- **PENDING（待确认）**：预约已提交，等待商家确认
- **CONFIRMED（已确认）**：商家已确认预约
- **COMPLETED（已完成）**：服务已完成
- **CANCELLED（已取消）**：预约已取消

## 管理员功能

### 1. 服务管理
- **服务项目管理**：
  - 添加服务：名称、分类、价格、时长、描述、图片
  - 编辑服务：修改服务信息
  - 删除服务：删除不需要的服务
  - 上架/下架：控制服务是否可预约
  
- **服务查询**：
  - 按分类筛选
  - 按状态筛选
  - 分页展示

### 2. 预约管理
- **预约列表**：
  - 查看所有用户的预约
  - 按状态筛选
  - 按日期范围筛选
  - 查看用户信息、宠物信息、服务信息
  
- **预约处理**：
  - 确认预约：将待确认状态改为已确认
  - 完成预约：将已确认状态改为已完成
  - 查看预约详情
  - 查看用户评价

### 3. 预约统计
- **统计数据**：
  - 总预约数
  - 待确认数量
  - 已确认数量
  - 已完成数量
  - 已取消数量
  - 总收入（已完成订单）
  
- **服务分类分布**：
  - 各类服务的预约数量统计
  - 可视化展示

## 技术实现

### 后端接口

#### 服务接口
- `GET /api/service/list` - 获取服务列表（分页、筛选）
- `GET /api/service/active` - 获取所有上架服务
- `GET /api/service/{id}` - 获取服务详情
- `POST /api/service` - 创建服务（管理员）
- `PUT /api/service` - 更新服务（管理员）
- `DELETE /api/service/{id}` - 删除服务（管理员）

#### 预约接口
- `POST /api/appointment` - 创建预约
- `PUT /api/appointment/cancel/{id}` - 取消预约
- `GET /api/appointment/{id}` - 获取预约详情
- `GET /api/appointment/my-list` - 获取我的预约列表
- `POST /api/appointment/review` - 添加评价
- `GET /api/appointment/admin/list` - 获取所有预约（管理员）
- `PUT /api/appointment/admin/status/{id}` - 更新预约状态（管理员）
- `GET /api/appointment/admin/statistics` - 获取预约统计（管理员）

## 数据库设计

### service 表（服务项目表）
```sql
- id: 服务ID
- name: 服务名称
- category: 服务分类（BATH/GROOM/FOSTER/MEDICAL）
- description: 服务描述
- price: 价格
- duration: 时长（分钟）
- image: 服务图片
- status: 状态（0-下架，1-上架）
- create_time: 创建时间
- update_time: 更新时间
```

### appointment 表（预约订单表）
```sql
- id: 预约ID
- order_no: 订单号
- user_id: 用户ID
- pet_id: 宠物ID
- service_id: 服务ID
- appointment_date: 预约时间
- status: 状态（PENDING/CONFIRMED/COMPLETED/CANCELLED）
- total_price: 总价
- notes: 备注
- rating: 评分（1-5）
- review: 评价内容
- create_time: 创建时间
- update_time: 更新时间
```

## 业务流程

### 预约流程
1. 用户浏览服务列表
2. 选择服务并点击"立即预约"
3. 选择宠物和预约时间
4. 提交预约，生成订单号
5. 预约状态为"待确认"
6. 管理员确认预约，状态变为"已确认"
7. 服务完成后，管理员标记为"已完成"
8. 用户可以对已完成的服务进行评价

### 取消流程
1. 用户在"我的预约"中查看预约
2. 对于"待确认"或"已确认"状态的预约
3. 点击"取消"按钮
4. 确认后预约状态变为"已取消"

### 评价流程
1. 预约状态为"已完成"
2. 用户点击"评价"按钮
3. 选择1-5星评分
4. 填写评价内容
5. 提交评价
6. 评价信息保存到预约记录中

## 特色功能

1. **订单号自动生成**：使用雪花算法生成唯一订单号
2. **预约时间验证**：不能预约过去的时间
3. **权限控制**：用户只能管理自己的预约
4. **状态流转**：严格的状态流转控制
5. **评价系统**：星级评分+文字评价
6. **统计分析**：多维度的预约数据统计

## 使用说明

### 普通用户
1. 登录系统后，点击"服务预约"
2. 浏览服务列表，选择需要的服务
3. 点击"立即预约"，填写预约信息
4. 在"我的预约"中查看预约状态
5. 服务完成后可以进行评价

### 管理员
1. 在"服务管理"中添加和管理服务项目
2. 在"预约管理"中查看所有预约
3. 确认用户的预约请求
4. 服务完成后标记为已完成
5. 查看预约统计数据
