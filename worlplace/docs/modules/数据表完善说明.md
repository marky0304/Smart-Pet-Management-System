## 数据插入方法

### 方法1：使用完整SQL文件（推荐）
```bash
# Windows
database\load_sample_data.bat

# 或手动执行
mysql -u root -p123456789 smart_pet_system < database\insert_sample_data.sql
```

### 方法2：分步插入
```bash
# 1. 插入宠物数据
mysql -u root -p123456789 smart_pet_system < database\insert_pets.sql

# 2. 插入其他数据
mysql -u root -p123456789 smart_pet_system < database\insert_sample_data.sql
```

### 方法3：通过MySQL客户端
```sql
USE smart_pet_system;
SOURCE database/insert_sample_data.sql;
```

## 注意事项

1. **外键约束**
   - 插入数据时需要注意表之间的依赖关系
   - 先插入父表数据，再插入子表数据
   - 顺序：user → pet → health_record, appointment, post, etc.

2. **数据编码**
   - 确保数据库和表使用utf8mb4编码
   - 支持中文和emoji表情

3. **密码加密**
   - 所有密码使用Hutool BCrypt加密
   - 不要使用在线工具生成的BCrypt哈希

4. **时间数据**
   - 使用合理的时间范围
   - 预约时间、订单时间等符合逻辑
