# 健康管理模块说明

## 功能概述

健康管理模块为宠物主人和管理员提供全面的宠物健康数据管理功能，包括健康记录管理、健康趋势分析、异常数据预警等。

## 用户功能

### 1. 健康记录管理
- **添加健康记录**：支持4种记录类型
  - 体检记录
  - 疫苗接种
  - 疾病记录
  - 体重记录
  
- **记录信息**：
  - 基础信息：宠物、记录类型、记录日期
  - 健康指标：体重、体温
  - 就医信息：症状、诊断、治疗方案
  - 医疗机构：医院名称、医生姓名
  - 用药信息：用药记录
  - 费用记录：医疗费用
  - 复诊提醒：下次复诊日期
  - 备注说明

### 2. 健康记录查询
- 按宠物筛选
- 按记录类型筛选
- 按日期范围筛选
- 分页展示
- 查看详情、编辑、删除

### 3. 健康趋势图表
- **体重趋势图**：折线图展示宠物体重变化
- **体温趋势图**：折线图展示体温变化，标注正常范围（37.5-39.5℃）
- 最近30条记录的趋势分析
- 可视化健康数据变化

### 4. 用药提醒
- 记录用药信息
- 下次复诊日期提醒

## 管理员功能

### 1. 健康数据统计
- **总记录数**：系统中所有健康记录总数
- **异常数据**：体温超出正常范围（<37.5℃ 或 >39.5℃）的记录数
- **待复诊数**：未来7天内需要复诊的记录数
- **记录类型分布**：饼图展示各类型记录的占比

### 2. 健康数据管理
- 查看所有宠物的健康记录
- 按类型、日期筛选
- 异常体温数据高亮显示（红色标注）
- 查看详细健康记录信息

### 3. 健康数据分析
- 记录类型分布统计
- 异常数据预警
- 复诊提醒统计

## 技术实现

### 后端接口

#### 用户接口
- `POST /api/health` - 创建健康记录
- `PUT /api/health` - 更新健康记录
- `DELETE /api/health/{id}` - 删除健康记录
- `GET /api/health/{id}` - 获取健康记录详情
- `GET /api/health/list` - 查询健康记录列表
- `GET /api/health/trend/{petId}` - 获取宠物健康趋势

#### 管理员接口
- `GET /api/health/admin/statistics` - 获取健康数据统计
- `GET /api/health/admin/list` - 获取所有健康记录

### 前端页面

#### 用户页面 (`/health`)
- 健康记录列表
- 添加/编辑/删除记录
- 健康趋势图表（ECharts）
- 筛选和搜索

#### 管理员页面 (`/health-management`)
- 健康数据统计面板
- 记录类型分布图（ECharts）
- 所有健康记录列表
- 异常数据标注

## 数据库设计

### health_record 表
```sql
- id: 记录ID
- pet_id: 宠物ID
- record_type: 记录类型（CHECKUP/VACCINE/ILLNESS/WEIGHT）
- record_date: 记录日期
- weight: 体重（kg）
- temperature: 体温（℃）
- symptom: 症状描述
- diagnosis: 诊断结果
- treatment: 治疗方案
- hospital: 医院名称
- doctor: 医生姓名
- medicine: 用药记录
- cost: 费用
- next_visit_date: 下次复诊日期
- notes: 备注
- create_time: 创建时间
- update_time: 更新时间
```

## 使用说明

### 普通用户
1. 登录系统后，点击左侧菜单"健康管理"
2. 点击"添加健康记录"按钮，填写宠物健康信息
3. 选择宠物后，可查看该宠物的健康趋势图
4. 可以编辑或删除自己的健康记录

### 管理员
1. 登录系统后，点击左侧菜单"健康数据管理"
2. 查看系统健康数据统计
3. 可以查看所有用户的健康记录
4. 关注异常数据和待复诊提醒

## 特色功能

1. **可视化趋势分析**：使用 ECharts 图表展示体重和体温变化趋势
2. **异常数据预警**：自动识别体温异常数据并标红显示
3. **复诊提醒**：统计未来7天内需要复诊的记录
4. **全面的健康档案**：记录宠物的体检、疫苗、疾病等各类健康信息
5. **权限控制**：用户只能管理自己宠物的健康记录，管理员可查看所有数据
